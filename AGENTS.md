# AGENTS.md

## Mission
Build and maintain the ACG AI Operations Platform as a production-grade SaaS product for SMB operations, AI-guided recommendations, modular installable business systems, workflow automation, knowledge management, analytics, and executive visibility.

## Product Definition
The platform helps small businesses:
- onboard and describe how they operate
- receive an AI-driven business analysis
- receive module and package recommendations
- install AI-powered operational systems
- connect tools and integrations
- run workflows
- measure outcomes in dashboards

## Core Product Loop
Marketing
-> Onboarding
-> Business Analysis
-> Recommendation Engine
-> Module / Package Installation
-> Workflow Activation
-> Dashboard Visibility
-> Continuous Optimization

## Engineering Priorities
1. Preserve architecture integrity
2. Keep modules config-driven
3. Keep recommendation logic explainable
4. Keep workflows auditable
5. Keep AI outputs structured and validated
6. Keep UI premium and cohesive
7. Prefer additive changes over rewrites
8. Preserve maintainability and testability
9. Register all modules cleanly in the module engine
10. Keep the product feeling like an operating system, not a generic tool

## Safety Rules
- Do not break existing flows
- Do not casually rewrite architecture
- Do not invent missing repo facts
- Do not expose secrets
- Validate before concluding a task
